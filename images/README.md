# 图片目录结构说明

此目录用于存放视频封面图片。

## 目录结构

```
public/images/
├── movies/        # 电影封面 (20张)
├── tv/            # 电视剧封面 (25张)
├── variety/       # 综艺封面 (20张)
```

## 文件命名规则

所有图片文件命名为 `{id}.jpg`，其中 `id` 对应 videos.js 中视频的 id。

## 下载图片步骤

1. 从各大视频平台(爱奇艺、优酷、腾讯视频、芒果TV)搜索对应的视频
2. 右键点击封面图片，选择"另存为"
3. 保存到对应的目录中，文件名按照上表命名
4. 图片建议尺寸: 280x420 (4:3 比例) 或 420x280 (3:2 比例)
5. 图片格式: JPG 或 PNG
